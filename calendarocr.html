CalendarOCR

An app that translates poster events from phone camera to calendar event and can be saved on device calendar. Ideal for college events.

Scroll

About the App

Built with React Native, CalendarOCR leverages Google Cloud Vision API for precise text extraction from event posters and GPT-4 API for intelligent understanding of event details. Simply snap a photo of any event poster, and the app instantly processes the information and saves it to your default calendar with all the relevant details.

See it in Action

Watch how CalendarOCR seamlessly transforms event posters into calendar entries. The app recognizes dates, times, locations, and event descriptions, creating perfectly formatted calendar events in seconds.

Key Features

Smart Text Extraction

Utilizes Google Cloud Vision API to accurately extract text from event posters, even with complex layouts or stylized fonts.

Intelligent Event Parsing

GPT-4 API intelligently understands event details from extracted text, identifying dates, times, locations, and descriptions with high accuracy.

One-Tap Calendar Integration

Seamlessly saves events to your device's default calendar app with all relevant information properly formatted and ready to use.

Perfect for College Life

Designed specifically for college students to quickly capture and organize campus events, lectures, club meetings, and social gatherings.

Technology Stack

Built with React Native for cross-platform mobile development. Integrates Google Cloud Vision API for OCR text extraction and GPT-4 API for natural language understanding. Native calendar integration for seamless event saving to device calendars.
